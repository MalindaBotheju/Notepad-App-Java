# 📝 Notepad App (Java)

## 📖 Overview
This is a simple **Notepad application** built using **Java Swing**.  
It allows users to:
- Create new text files
- Open existing files
- Edit and save text
- Exit the application

This project was developed as part of an **Advanced Programming Techniques** assignment.

---
